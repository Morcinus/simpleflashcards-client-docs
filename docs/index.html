Home

Tato dokumentace je zaměřena na samotnou webovou aplikaci, popis jednotlivých komponentů a funkcí, které komunikují s API serveru.

Rozdělení Projektu Simpleflashcards:

Projekt je rozdělen do dvou částí:

  • Cloud Functions - API, které bylo vytvořeno pro komunikaci mezi Cloud Firestore databází a klientem.
  • Samotná Simple Flashcards webová aplikace

Tato dokumentace je zaměřena na webovou aplikaci. Dokumentaci pro Cloud Functions můžete najít zde.

Aplikace

Tento projekt je vytvořen pomocí těchto nejpodstatnějších knihoven: React, Redux, Axios, Material UI. Od toho se odvíjí struktura celé aplikace.

React

Aplikace je sestavena z jednotlivých komponentů. Popis každého komponentu aplikace můžete najít v této dokumentaci.

Redux

Tato knihovna je klíčovou součástí této aplikace. Umožňuje přehledné předávání informací mezi danými komponenty aplikace. Díky této knihovně je vytvořen tzv. store. Na tomto místě jsou uložena data aplikace. Když potřebuje např. nějaký komponent tato data získat, stačí mu přistoupit do storu a nemusí je tak získávat od ostatních komponentů.

Axios

Axios je knihovna, která zajišťuje komunikaci mezi klientem a serverem. Využívají ji funkce této aplikace pro komunikaci s API serveru.

Material UI

Samotný vzhled aplikace je zajištěn pomocí knihovny Material UI. Tato knihovna umožňuje rychlé a přehledné vytváření komponentů s jednotným vzhledem.

Struktura dokumentace

Projekt je rozdělen do těchto skupin:

  • Components - Zde můžete najít jednotlivé komponenty této aplikace.
  • Pages - Zde můžete najít komponenty aplikace, které tvoří danou stránku. K tomu využívají komponenty se složky components.
  • ReduxActions - Zde můžete najít veškeré funkce, které komunikují s API serveru a s Redux reducery (čímž mění hodnoty ve Redux store).
  • ReduxReducers - Zde můžete najít všechny Redux Reducery.
  • Util - Zde můžete najít Javascriptové moduly, které jsou používány ostatními částmi aplikace.
